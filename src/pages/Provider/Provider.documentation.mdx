# Provider

The provider page allow to the member see the details of a specific provider,
this page is mixed with the Shopping Cart since once the user enter on this Page,
he could start to add products to heir Shopping Cart.

Once the user exit from the Provider Page, the Shopping Cart will be cleared.

Shopping cart is individual by each Provider
